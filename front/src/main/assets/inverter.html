      1. Theoretical  background for the experiment /Validation of the experiment:

Operational amplifiers are high gain integrated circuit amplifiers with two high-impedance                                  input terminals and one low-impedance output. The inputs are identified as noninverting input and inverting input because of the way they affect the output voltage. The basic circuit of an operational amplifier consists of a differential amplifier input stage and an emitter-follower output stage. By the use of externally connected resistors, an operational amplifier can be employed for a great many applications. The simplest of these are the voltage follower, the non-inverting amplifier and the inverting amplifier.
Voltage follower:
The voltage follower or unity-gain buﬀer is a unity-gain non-inverting ampliﬁer. The inverting input terminal is connected directly to the output terminal and the signal is applied to the noninverting input terminal. The output voltage follows the input, giving the circuit a gain of one. The voltage follower has very high input impedance and very low output impedance.
To produce an output closely equal to Vd =V i /M
Where M is the internal voltage gain (open-loop gain) of the operational amplifier. The precise output voltage is
Vo=Vi-Vd
Vo=Vi-(Vi/M)
Where Vi/M is an error voltage.

Non-inverting amplifier:
Non-inverting amplifier behaves in a similar way to the voltage follower circuit. The difference is, instead of all of the output being fed directly back to the input, only a portion is fed back. The output voltage is potentially divided across resistors R2 and R3 before it is applied to the inverting input.
VR3=Vi=I2R3
V0=I2(R2+R3)
Av = Vo/Vi =I2(R2+R3)/I2R3
Av=(R2 +R3)/R3

Inverting amplifier:
The inverting amplifier is so named, because an input applied via R1 to the inverting input terminal causes the output to go in a negative direction when the input is positive-going.
The input and output voltages can be expressed as
Vi=I1R1
Vo= -I1R2
Av=Vo/Vi=(-I1 R2)/( I1R1)
Av= -R2/R1

7.3.7     Step by step procedure to carry out the experiment:

  1. Connect the circuit for inverting, non-inverting amplifier and voltage follower on breadboard.
  2. Connect the input terminal of the op amp to the function generator and the output terminal to the CRO.
  3. Feed input from the function generator and observe the output on the CRO.
  4. Observe the input and the output waveforms.
